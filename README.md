# Unnamed Game
I have no clue what this game is going to be yet :)

This game is most likely going to end up as an indie-platformer. However, we'll have to see how it goes. It currently features a simple box with a black background and a simple square-character.

The game only currently features a single level, with 4 platforms.

## Playing the game
### Controls
 - Up/W/SPACE  =  Jump
 - Left/A      =  Move left
 - Right/D     =  Move Right
